## 📖 درباره پروژه

**آوای یقین** یک دستیار هوشمند دینی است که با استفاده از **Groq API** و مدل **Mixtral-8x7b** پاسخگوی سوالات دینی کاربران است.

### ✨ ویژگی‌ها

- 🕌 **طراحی شیشه‌ای** با المان‌های مسجدی
- 🤖 **پاسخگویی هوشمند** با استفاده از AI
- 👤 **سیستم کاربری** با ذخیره‌سازی چت‌ها
- ⚙️ **پنل مدیریت کامل** با امکانات:
  - تغییر نام ربات
  - تغییر شخصیت ربات
  - تغییر پیام خوش‌آمدگویی
  - مشاهده چت‌های کاربران
  - آمار کاربران و پیام‌ها
  - خروجی JSON/CSV/Excel
- 🔐 **ورود ادمین** با رمز مخصوص
- 💾 **ذخیره‌سازی SQLite**
